 场景： 一张静态图 - 运动起来 -- 合成单个视频 -- 多张单个视频合成完整视频, 适用于 小说类的视频剪辑
 
 > 脚本已经写好， 请勿乱修改

 ## 文件名称解释：

 1. imgs： 静态图片文件夹， 将你的照片丢到imgs文件夹下， 目前是 jpg 格式， 如果需要png 请修改 `2_every_img_video.sh`的jpg为png
 2. result： 输出结果
 3. temp： 临时存放文件夹
